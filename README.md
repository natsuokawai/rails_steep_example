Rails with RBS and Steep.

## Generate RBS files
You can generate RBS files using [rbs_rails](https://github.com/pocke/rbs_rails).

Rails:
```
$ bundle exec rake copy_signature_files
```

Model classes:
```
$ bundle exec rake generate_rbs_for_model
```
